## Contributing
To contribute assets to the **Soiri Lab** channel, please ensure you follow these steps:

* **Read the License:** Please review the [License](/LICENSE) to understand how these assets can be used.
* **Review the Guidelines:** Read the [Guidline](/CONTRIBUTING.md) and understand it before any submission.

**Note:** All pull requests must adhere to these documents to be considered for merge.
